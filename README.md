# Design4Sales · Интерактивный контент-календарь

Мини-приложение на React/Vite для работы с календарём публикаций Design4Sales за период `27.04.2026 – 25.07.2026`.

## Как запустить локально

```bash
npm install
npm run dev
```

После запуска откройте локальный адрес, который покажет Vite.

## Как собрать проект

```bash
npm run build
```

Готовые статические файлы появятся в папке `dist/`.

## Как выложить на GitHub Pages

Вариант через GitHub Pages без CLI:

1. Загрузите папку проекта в GitHub-репозиторий.
2. Выполните `npm install` и `npm run build`.
3. В настройках репозитория откройте `Settings → Pages`.
4. Выберите публикацию из ветки или через GitHub Actions.
5. Если используете ветку `gh-pages`, можно установить зависимость и запустить:

```bash
npm run deploy
```

## Прогресс и статусы

Прогресс задач хранится в `localStorage` браузера. Для переноса на другой компьютер используйте:

- `Экспорт прогресса JSON` — скачивает файл с текущими статусами.
- `Импорт прогресса JSON` — загружает ранее сохранённый файл.
- `Сбросить прогресс` — очищает все отметки в текущем браузере.

## Где лежат данные календаря

Основные данные находятся в:

```text
src/data/calendarData.js
```

Каждая дата содержит:

- `priority` — главное;
- `optional` — дополнительно;
- `archive` — архив / перепаковка;
- `prepare` — что подготовить;
- `cta` — цель или действие.

Данные были структурированы из карточного календаря Design4Sales.
